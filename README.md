### TDwnld

|        |Versions|
|--------|--------|
| pip    | 25.0   |
| python | 3.13.1 |

Install the dependencies
```
pip install -r requirements.txt
```

#### Prerequesites:
+ You need an API key from [OMDb API](https://www.omdbapi.com/apikey.aspx "https://www.omdbapi.com/apikey.aspx") and apply for a key
+ You might have to wait up to 1 hour for the email containing key
+ create a text file "API_key.txt" and type the key in the file with no spaces and save

Usage:
```
python downloader.py
```
